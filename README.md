# MUTIMODEL-Knee-Osteoarthritis

A comprehensive multimodal approach for knee osteoarthritis diagnosis using radiographic imaging and clinical/tabular patient data.

## Contents

- `Mutimodel knee.py`: primary model training and evaluation script
- `.gitignore`: ignores local environment, temporary files, and unrelated workspace folders

## Notes

This repository is configured to track only the project files relevant to the knee osteoarthritis model and to exclude unrelated portfolio directories present in the local workspace.
